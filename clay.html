Clay
3D Object Modeler
Clay lets you sculpt 3D shapes by drawing silhouettes on six faces of a cube.

Draw on each face — transparent areas get carved away. What remains becomes your 3D object, textured from your drawings.

The idea was invented at age 7 by the creator of this app. It uses a computer vision technique called visual hull reconstruction — the same method used in professional motion capture and 3D scanning systems.

"The visual hull of an object is the shape carved out of space by projecting the object's silhouette from multiple viewpoints. It is the largest possible shape consistent with all observed silhouettes, and serves as an upper bound on the true object geometry."
— based on Laurentini (1994), IEEE TPAMI

How to use Clay

1

Add an Object

Click + New in the Objects panel to create a new object. Click its name (while selected) to rename it. Use Dup to copy it and Del to delete.

2

Draw on the Faces

Click any face thumbnail to open the face editor. Draw your shape's silhouette.

Opaque pixels = kept. Transparent = carved away.

Use Clear (transparent) to start fresh, then draw a shape. The Disk tool on all 6 faces makes a sphere!

Tip: Use Import PNG in the editor to load a real image as a face texture. Use Export PNG to save a face to edit externally.
3

Build the Shape

Click ▶ Build Selected. Use the Res slider for quality — higher = more detail, slower. Start at 32.

4

Navigate the 3D View

Left-drag to rotate. Scroll to zoom. Right-drag to pan.
Hold G + drag to grab and move the selected object. Add Shift to move up/down.

5

Resize Objects

Use the Scale fields in the Transform panel to resize. Enable Uniform scale to scale all axes together by typing in any field.

6

Export

Click Export OBJ to download your scene as a .obj file for Blender, Tinkercad, or any 3D software.

Sphere recipe: Open each face → Clear (transparent) → Disk tool → draw a large circle filling the canvas → Done. Repeat all 6 faces then Build!
